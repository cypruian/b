Introducing API

Bitcoin live and historical data is available through a unified REST API. Features of the new v0.2.1 API:
  • Unified endpoints
  • Simple interface for all the endpoints
  • Full-history data from genesis block
  • Private “api_key” header parameter
  • Custom data format
  • Hourly and daily frequencies
  • Scalable and robust AWS data infrustracture
Contact us at contact@cypruian.com if you would like to get access to the v0.2.1 API and build a custom plan according to your data requirements.
